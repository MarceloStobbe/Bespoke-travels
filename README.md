This is a personal travel project created to showcase curated travel experiences. The website highlights unique destinations including cities, beaches, and mountains.

✨ Features
* **Modern Design**: A clean and professional look.
* **Responsive**: Works on mobile, tablets, and desktops.
* **Interactive**: Includes a photo carousel and a contact form.

🛠️ Built With
* **HTML5**
* **CSS3** (Custom styles)
* **Bootstrap 5** (Layout and components)

## 📂 Project Structure
* `index.html`: The main homepage.
* `about.html`: Information about the project.
* `Contact.html`: Contact form page.
* `/css`: Folder containing the custom stylesheet.
* `/img`: Folder containing travel photos and logos.

## 🚀 How to Run
1. Download or clone this repository.
2. Open the `index.html` file in your web browser.
